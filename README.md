# PyTube GUI Shri

Provides a simple PyQt-based GUI front-end to [PyTube](https://github.com/get-pytube/pytube3/)

## Usage

```
pytube-gui-shri [<url>]
```

## Dependencies

1. Python 3
2. PyTube
3. PyQt 5
4. ffmpeg (for merging separately selected video and audio streams)
